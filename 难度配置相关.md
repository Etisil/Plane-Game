# 游戏难度配置说明

本文档只记录和游戏难度直接相关的配置项，方便快速调整普通遭遇战、Boss 战、弹幕密度和玩家容错。

## 1. 难度选择入口

位置：`plane_main.py`

```python
PlaneGame.__select_difficulty()
```

这里负责显示难度选择界面，并返回：

```python
"normal"
"hard"
"lunatic"
```

当前界面会显示：

- 普通
- 困难
- 疯狂

如果要修改难度名称、说明文字或增加新难度，优先改这里的 `options`：

```python
options = [
    ("normal", "普通", "生命20  简单首领"),
    ("hard", "困难", "生命15  普通首领"),
    ("lunatic", "疯狂", "生命10  高难首领"),
]
```

注意：新增难度时，还需要同步修改 `plane_sprites.py` 中的生命和 Boss 弹幕倍率配置。

## 2. 玩家生命配置

位置：`plane_sprites.py`

```python
DIFFICULTY_HERO_HP = {
    "normal": 20,
    "hard": 15,
    "lunatic": 10,
}
```

含义：

- `normal`：普通难度玩家 HP。
- `hard`：困难难度玩家 HP。
- `lunatic`：疯狂难度玩家 HP。

数值越大，玩家容错越高。

例如想让普通难度更简单：

```python
"normal": 25
```

## 3. Boss 普通弹幕频率

位置：`plane_sprites.py`

```python
BOSS_INTERVAL_SCALE = {
    "normal": 2.9,
    "hard": 1.7,
    "lunatic": 1.0,
}
```

这个配置控制 Boss 普通弹幕的发射间隔倍率。

实际发射间隔：

```text
实际间隔 = 基础间隔 * BOSS_INTERVAL_SCALE[当前难度]
```

数值越大，弹幕越稀疏；数值越小，弹幕越密。

例如：

```python
self.__on_pattern(10)
```

在不同难度下大约变成：

```text
normal: 10 * 2.9 = 29 帧一次
hard:   10 * 1.7 = 17 帧一次
lunatic:10 * 1.0 = 10 帧一次
```

如果想提高 `hard` 难度 Boss 弹幕密度，可以改成：

```python
"hard": 1.4
```

如果想降低 `normal` 难度 Boss 弹幕密度，可以改成：

```python
"normal": 3.3
```

## 4. Boss 全屏弹幕额外频率

位置：`plane_sprites.py`

```python
BOSS_FULLSCREEN_INTERVAL_SCALE = {
    "normal": 1.2,
    "hard": 1.0,
    "lunatic": 1.0,
}
```

这个配置只影响全屏压制型弹幕，也就是使用：

```python
self.__on_fullscreen_pattern(...)
```

的弹幕。

实际发射间隔：

```text
实际间隔 = 基础间隔 * BOSS_INTERVAL_SCALE[难度] * BOSS_FULLSCREEN_INTERVAL_SCALE[难度]
```

例如：

```python
self.__on_fullscreen_pattern(12)
```

在 `normal` 下约为：

```text
12 * 2.9 * 1.2 = 41.76 帧一次
```

在 `lunatic` 下为：

```text
12 * 1.0 * 1.0 = 12 帧一次
```

如果觉得普通难度全屏弹幕仍然太密，可以提高：

```python
"normal": 1.5
```

如果想让困难难度全屏弹幕更强，可以降低：

```python
"hard": 0.85
```

## 5. Boss 每阶段血量

位置：`plane_sprites.py`

```python
BOSS_PHASE_HP = (100, 130, 140, 180, 220)
```

五个数字分别对应 Boss 第 1 到第 5 阶段。

```text
第1阶段: 100
第2阶段: 130
第3阶段: 140
第4阶段: 180
第5阶段: 220
```

数值越大，该阶段持续越久。

例如想加强后期：

```python
BOSS_PHASE_HP = (100, 130, 160, 220, 280)
```

## 6. Boss 转阶段休息时间

位置：`plane_sprites.py`

```python
BOSS_PHASE_BREAK_FRAMES = 90
BOSS_PHASE_FILL_FRAMES = 90
```

含义：

- `BOSS_PHASE_BREAK_FRAMES`：阶段击破后的闪烁/等待时间。
- `BOSS_PHASE_FILL_FRAMES`：新阶段血条从 0 填满的时间。

数值越大，玩家转阶段休息时间越长。

例如增加休息时间：

```python
BOSS_PHASE_BREAK_FRAMES = 120
BOSS_PHASE_FILL_FRAMES = 120
```

## 7. Boss 阶段弹幕配置位置

位置：`plane_sprites.py`

函数：

```python
Boss.fire_danmaku()
```

阶段对应关系：

```python
if self.phase == 0:
    # 第1阶段
elif self.phase == 1:
    # 第2阶段
elif self.phase == 2:
    # 第3阶段
elif self.phase == 3:
    # 第4阶段
else:
    # 第5阶段
```

### 普通弹幕触发

普通弹幕通常使用：

```python
self.__on_pattern(间隔, 偏移)
```

例：

```python
if self.__on_pattern(8):
    bullets.extend(...)
```

`8` 越小，发射越频繁。

### 全屏弹幕触发

全屏弹幕通常使用：

```python
self.__on_fullscreen_pattern(间隔, 偏移)
```

例：

```python
if self.__on_fullscreen_pattern(12):
    bullets.extend(...)
```

该间隔会同时受：

- `BOSS_INTERVAL_SCALE`
- `BOSS_FULLSCREEN_INTERVAL_SCALE`

影响。

### 全屏弹幕分组轮换

当前部分全屏弹幕使用：

```python
self.__fullscreen_group_active(round_interval, group_index, group_count, rounds_per_group)
```

含义：

- `round_interval`：一轮的基础长度。
- `group_index`：当前弹幕属于第几组。
- `group_count`：总共有几组。
- `rounds_per_group`：每组连续持续几轮。

例：

```python
self.__fullscreen_group_active(16, 0, 3, 3)
```

表示：

- 以 `16` 为基础轮长。
- 总共 3 组全屏弹幕。
- 当前判断第 0 组。
- 每组连续发射 3 轮后切换到下一组。

如果想让全屏弹幕切换更快，可以减少 `rounds_per_group`：

```python
self.__fullscreen_group_active(16, 0, 3, 2)
```

如果想让每组持续更久，可以增加它：

```python
self.__fullscreen_group_active(16, 0, 3, 4)
```

## 8. 遭遇战敌机生成难度

位置：`plane_main.py`

### 普通敌机生成频率

```python
pygame.time.set_timer(CREATE_ENEMY_EVENT, 1800)
```

数值越小，敌机刷新越快。

例：

```python
pygame.time.set_timer(CREATE_ENEMY_EVENT, 1500)
```

会让遭遇战敌机更多。

### 普通敌机发弹频率

```python
pygame.time.set_timer(ENEMY_FIRE_EVENT, 1400)
```

数值越小，普通敌机发弹越频繁。

### 高级敌机出现概率

```python
enemy = Enemy(2 if random.randint(0, 100) < 45 else 1)
```

这里的 `45` 约表示 45% 概率生成中型敌机。

提高它可以增加高级敌机：

```python
enemy = Enemy(2 if random.randint(0, 100) < 60 else 1)
```

降低它可以减少高级敌机：

```python
enemy = Enemy(2 if random.randint(0, 100) < 25 else 1)
```

## 9. Boss 出现时间

位置：`plane_main.py`

```python
if self.score.getvalue() > 120 + 420 * self.index:
```

含义：

- `120`：首次 Boss 出现分数。
- `420`：后续 Boss 出现分数递增量。

当前游戏通常只需要关注首次 Boss，因此主要调 `120`。

更早进入 Boss：

```python
if self.score.getvalue() > 80 + 420 * self.index:
```

更晚进入 Boss：

```python
if self.score.getvalue() > 180 + 420 * self.index:
```

## 11. 弹幕性能难度

位置：`plane_main.py`

```python
MAX_DANMAKU_BULLETS = 650
DANMAKU_REMOVE_RATE = 0.1
```

含义：

- `MAX_DANMAKU_BULLETS`：屏幕允许存在的最大弹幕数量。
- `DANMAKU_REMOVE_RATE`：超出上限后每次清理比例。

如果机器性能较弱，可以降低上限：

```python
MAX_DANMAKU_BULLETS = 450
```

如果想让高难度弹幕更完整，可以提高上限：

```python
MAX_DANMAKU_BULLETS = 800
```

## 12. 推荐调整顺序

如果只是调难度，建议按这个顺序：

1. 调整 `DIFFICULTY_HERO_HP`。
2. 调整 `BOSS_INTERVAL_SCALE`。
3. 调整 `BOSS_FULLSCREEN_INTERVAL_SCALE`。
4. 调整 `BOSS_PHASE_HP`。
5. 调整 `Boss.fire_danmaku()` 中单阶段的具体弹幕间隔。
6. 最后再调整遭遇战敌机生成频率和高级敌机概率。

这样可以避免一次改太多导致难度失控。
